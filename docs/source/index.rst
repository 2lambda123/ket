Welcome to Ket's documentation!
===============================

Ket is a quantum programming framework for dynamic interaction between
classical end quantum computer that features:

* Ket, a Python-embedded classical-quantum programming language;
* Libket, a quantum programming libary for C++ and Python;
* Ket Bitwise Simulator (KBW) for quantum circuit simulator with bitwise
  operation [arXiv:2004.03560]_.

Source code
-----------

* Ket interpreter and libket for Python: https://gitlab.com/quantum-ket/ket
* libket for C++: https://gitlab.com/quantum-ket/libket
* Ket Bitwise Simulator: https://gitlab.com/quantum-ket/kbw
